# Data Science Tutorial Repository

This repository contains a collection of machine learning tutorials and example projects covering several core topics:

- **Introduction to Neural Networks**: notebooks and Python scripts demonstrating simple neural networks, linear/quadratic fitting, cluster classification, and more.
- **Image Classification**: example model and notebook for image-based deep learning.
- **Mall Customer Segmentation**: a notebook exploring customer segmentation using clustering techniques.
- **Predicting Healthcare Cost**: data preparation and modeling examples for healthcare cost prediction.
- **PySpark Tutorial**: PySpark notebooks and sample CSV data for learning distributed data processing.

Each folder includes code, datasets, and notebooks to help learners explore common data science workflows.
